# Learn Platform - حالة المشروع

## 📊 ملخص التطوير

منصة تعليمية عالمية مدعومة بالذكاء الاصطناعي، تهدف إلى توفير خطط دراسية مخصصة لكل متعلم.

---

## ✅ المراحل المكتملة

### المرحلة الأولى: الوضع الداكن وتعدد اللغات ✓
- **Dark Mode**: تم إضافة زر تبديل الوضع الداكن/الفاتح
- **تعدد اللغات (i18n)**:
  - العربية (ar)
  - الإنجليزية (en)
  - الفرنسية (fr)
- **الميزات**:
  - اكتشاف اللغة تلقائياً من المتصفح
  - حفظ التفضيلات في localStorage
  - تبديل اتجاه الصفحة تلقائياً (RTL/LTR)

### المرحلة الثانية: واجهة AI Agent ✓
- **صفحة جديدة**: `/ai-agent`
- **واجهة محادثة تفاعلية**:
  - نظام أسئلة ذكية (4 أسئلة)
  - عرض الخطة الدراسية المقترحة
  - قائمة الدورات مع المدة والمستوى
  - معالومات عن الأهداف والمراحل
- **التصميم**:
  - متجاوب (Responsive)
  - حركات سلسة (Smooth Animations)
  - دعم الوضع الداكن

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **Vite** - أداة البناء السريعة
- **TypeScript** - للأمان النوعي
- **Tailwind CSS** - للتصميم
- **Radix UI** - مكونات واجهة المستخدم
- **Framer Motion** - الحركات والتأثيرات
- **i18next** - تعدد اللغات
- **React Query** - إدارة البيانات

### Backend
- **Node.js + Express** - خادم الويب
- **PostgreSQL** - قاعدة البيانات
- **Drizzle ORM** - مدير قاعدة البيانات
- **Passport.js** - المصادقة
- **Stripe** - معالجة الدفع

### AI Integration (قريباً)
- **OpenAI API** - لإنشاء الخطط الدراسية

## 🚀 كيفية التشغيل

### التثبيت والإعداد
```bash
# تثبيت التبعيات
pnpm install

# إعداد قاعدة البيانات
npx drizzle-kit push

# تعبئة البيانات التجريبية
npx tsx server/db/seed.ts
```

### التشغيل
```bash
# تشغيل الخادم والعميل
npm run dev

# الخادم: http://localhost:3001
# العميل: http://localhost:3001 (مدمج)
```
